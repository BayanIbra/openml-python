    openml.datasets.list_datasets¶

    openml.datasets.list_datasets(offset=None, size=None, tag=None)¶

    Return a list of all dataset which are on OpenML.

    Parameters:

    offset : int, optional

    the number of datasets to skip, starting from the first

    size : int, optional

    the maximum datasets of tasks to show

    tag : str, optional

    the tag to include
    Returns:

    datasets : list of dicts

    A list of datasets having the given tag (if applicable).

    Every dataset is represented by a dictionary containing the following information: - dataset id - status

    If qualities are calculated for the dataset, some of these are also returned.
